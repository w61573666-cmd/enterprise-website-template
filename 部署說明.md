## 二、部署步驟

### 方法A：FTP 上傳（推薦新手）

1. **登錄阿里雲控制台** → 輕量應用服務器 → 找到你的服務器
2. **查看 FTP 信息**：應用管理 → 網站設置 → FTP 賬號密碼
3. **使用 FTP 客戶端**（FileZilla / Transmit）連接：
   - 主機：服務器 IP
   - 用戶名/密碼：控制台顯示的 FTP 賬號
   - 端口：21
4. **上傳文件**：
   - 將 `hsst/` 文件夾內所有內容上傳到 `/www/wwwroot/你的網站目錄/`
   - 通常路徑是 `/www/wwwroot/你的域名/` 或 `/var/www/html/`

### 方法B：SSH 部署（推薦進階用戶）

```bash
# 1. SSH 登錄服務器
ssh root@你的服務器IP

# 2. 創建網站目錄
mkdir -p /var/www/hsst

# 3. 上傳文件（在本地 Mac 終端執行）
scp -r /Users/stone/Desktop/hsst/* root@你的服務器IP:/var/www/hsst/

# 4. 設置權限（在服務器執行）
chmod -R 755 /var/www/hsst
chown -R www-data:www-data /var/www/hsst
```

## 三、Nginx 配置（如需）

如果你的服務器使用 Nginx：

```nginx
server {
    listen 80;
    server_name www.hsst.hk hsst.hk;

    root /var/www/hsst;
    index index.html;

    # 中文頁面
    location / {
        try_files $uri $uri/ =404;
    }

    # 英文頁面
    location /en/ {
        try_files $uri $uri/ =404;
    }

    # 啟用 gzip
    gzip on;
    gzip_types text/html text/css application/javascript image/svg+xml;
}
```

重啟 Nginx：
```bash
nginx -t          # 測試配置
nginx -s reload   # 重啟
```

## 四、SSL 證書（HTTPS）

### 免費證書申請
1. 阿里雲控制台 → SSL 證書 → 購買證書 → 選擇「免費證書」
2. 填寫域名 `hsst.hk` 和 `www.hsst.hk`
3. DNS 驗證（添加一條 TXT 記錄）
4. 下載證書後配置 Nginx

### Nginx HTTPS 配置
```nginx
server {
    listen 443 ssl;
    server_name www.hsst.hk hsst.hk;

    ssl_certificate /etc/nginx/ssl/hsst.hk.pem;
    ssl_certificate_key /etc/nginx/ssl/hsst.hk.key;

    root /var/www/hsst;
    # ... 其他配置
}

# HTTP 跳轉 HTTPS
server {
    listen 80;
    server_name www.hsst.hk hsst.hk;
    return 301 https://$host$request_uri;
}
```

## 六、常見問題

### Q: 圖片無法顯示？
A: 檢查 images 文件夾是否完整上傳，權限是否正確

### Q: CSS 樣式失效？
A: 檢查 css 文件夾路徑是否正確，文件名是否完整

### Q: 表單無法提交？
A: Formspree 是第三方服務，無需服務器配置，只要 HTML 文件正確即可
