廣州國際會展中心石材工程

廣州 · 大型公共基礎設施石材方案

🏥 項目類型：醫療與公共空間 🪨 石材品類：廣西白花崗岩 / 鈦金洞石 / 銀影石 ⚙️ 加工工藝：CNC切割 / 水刀切割 / 抗菌塗層 ✨ 表面處理：啞光 / 火燒 / 抗菌納米 📦 供貨週期：12-14 週 🔧 安裝週期：20 週 🏆 地標項目：是 🏅 獲獎情況：2023 中國建築工程魯班獎
Project Overview

項目概覽

廣州國際會展中心是華南地區最大的會展場地之一，佔地面積逾80萬平方米。恆生石材為其提供80,000平方米的大型石材工程，採用廣西白花崗岩、鈦金洞石及銀影石等材料，並進行抗菌納米塗層處理和耐火板材測試，確保公共空間的安全與美觀。

Technical

技術亮點與解決方案

恆生石材以專業技術和豐富經驗，為每一個項目攻克難關。

🦠
公共衛生
抗菌納米塗層
會展中心人流密集，衛生要求嚴格。所有室內石材均採用食品級安全抗菌納米塗層，對大腸桿菌和金黃葡萄球菌的抑制率高達99.9%，同時不影響石材的自然觸感和外觀。
🔥
消防安全
耐火石材測試
會展中心屬於大型公共建築，耐火等級要求極高。所用石材全部通過ASTM E119耐火極限測試，石材背襯採用A1級防火材料，確保在緊急情況下的結構完整性。
📐
施工管理
80,000㎡大型排版管理
超大面積施工採用分區管理模式，將整個會展中心劃分為12個施工區段，每區段配備獨立測量團隊，確保工期按計劃完成，總工期縮短20%。
